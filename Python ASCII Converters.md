# Python-ASCII-Converters
 - Includes 5 Scripts that convert Videos into various ASCII Animations

# Adaptive
Run the Install Dependencies file

Put your video in the MEDIA folder

Run the Start script

Frames will render, pygame will open, but playback wont start until all frames and surfaces have been rendered
This ensures a smooth playback at the requested speed

After the playback is done, leave the window open, for the script to save the ascii as a video

The ascii video will be saved in the OUTPUT folder

You can change the settings of it in the settings json
